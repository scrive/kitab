Thanks for your interest in Kitab!

## Development Environment

Kitab is developed in Haskell. You can get the appropriate compiler version from [GHCup](https://www.haskell.org/ghcup/) or your system's package manager.

We use the following tools:

* `fourmolu-1.19.0.1`
* `hlint-3.10` (available on GHCup)
* `cabal-gild-1.5.0.3`
* `scripths-0.5.2.0`

You can install them with `cabal install -j <name>-<version>` or use GHCup to get them when available.

## Submitting changes

We track changes through issues and pull requests.
We require that you open a ticket that explains the problem and/or the solution before
you open a pull request.
